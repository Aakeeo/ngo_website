# Reminiscence Welfare Trust (RWT) — Website

A professional, mobile-optimized single-page website for the Reminiscence Welfare Trust (RWT), showcasing their impactful work in healthcare, education, and community welfare.

## Features

✅ **Modern Design** - Clean, professional UI with smooth animations
✅ **Mobile Responsive** - Fully optimized for all screen sizes
✅ **Gallery Section** - Showcasing RWT's activities with images from PowerPoint
✅ **Color Palette** - Matches official RWT logo colors (Blue #156ea4 & Lime #c8d835)
✅ **Contact Information** - Complete contact details for President & Secretary
✅ **Donation Information** - Bank details with 80G tax exemption notice
✅ **Fast Loading** - Optimized images with lazy loading
✅ **Accessible** - Semantic HTML and ARIA labels

## Sections

1. **Hero** - Eye-catching introduction with CTA
2. **About** - How RWT started during COVID-19
3. **Impact** - Key statistics and achievements
4. **Our Work** - Detailed breakdown of initiatives
5. **Gallery** - Photo gallery of activities with captions
6. **Contact** - Full contact information and donation details
7. **Footer** - Quick links and copyright

## Deployment

### GitHub Pages
1. Create a new repository
2. Upload all files
3. Enable GitHub Pages in repository settings
4. Your site will be live at `https://yourusername.github.io/reponame`

### Netlify (Drag & Drop)
1. Visit [netlify.com](https://netlify.com)
2. Drag the entire folder
3. Your site will be deployed instantly

### Traditional Web Hosting
Upload all files to your hosting provider's public_html or www folder.

## Customization

### Update Content
- Edit `index.html` for text content
- Modify `assets/styles.css` for styling changes
- Update colors in CSS `:root` section

### Add More Gallery Images
1. Place images in `assets/gallery/`
2. Add new gallery items in the `#gallery` section of `index.html`
3. Follow the existing HTML structure
